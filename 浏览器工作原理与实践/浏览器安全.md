### 网页安全

- 同源策略
  - DOM 层面
    - window.open 或 a 标签
    - window.opener
  - 数据层面
    - cookie
    - storage
    - indexDB
  - 网络层面
    - XMLHttpRequest
- 浏览器出让的同源策略
  - 页面中可以嵌入第三方资源
    - XSS 攻击与 CSP（内容安全策略）
  - 跨域资源共享（CORS）
  - 跨文档消息机制
    - window.postMessage

```java
// CSP
Content-Security-Policy: default-src 'self' *.trusted.com
```
